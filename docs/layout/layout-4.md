## 思路

CSS 值与单位（第三版）（http://w3.org/TR/css-values-3/#calc） 定义了一个 calc() 函数， 它允许我们在 CSS 中直接以这种简单的算式来指定属性的值。

:::tip
别忘了在 `calc()` 函数中用空白符把 - 和 + 隔起来，否则会产生解析错误！这个看似怪异的规则是为了向前兼容：在未来，calc() 可能会允许使用标识符，而这些标识符本身可能会包含连字符，容易与减号混淆。
:::
